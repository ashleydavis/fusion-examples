# fusion-examples

Examples of using [Fusion](https://www.npmjs.com/package/@codecapers/fusion) for DI in React class components and functional components.

Clone or download this repo.

There are two examples, so `cd class-components` or `cd functions-components` depending on what interests you.

Install dependencies:

```bash
npm install
```

Run the code:

```bash
npm start
```

Open your browser and navigate to http://localhost:1234/.

## Get Fusion

- [npm](https://www.npmjs.com/package/@codecapers/fusion)
- [GitHub](https://github.com/ashleydavis/fusion)
